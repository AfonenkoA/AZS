Определение списка организаций, которым принадлежат АЗС общего пользования из документа “Программа развития сети автозаправочных станций в Республике Беларусь на 2016–2020 годы”.  

Самые актуальные данные будут на сайтах компаний. После проверки сайтов компаний на наличие списка АЗС оказалось, что большинство сайтов компаний предоставляют карту с обозначенными заправками или динамически загружаемый список. Такой формат данных не подходит для автоматического анализа полученной http запросом страницы. 
Вариант — это использовать API Google или Yandex карт, но сохранение данных поисковых запросов и преобразование адреса в координаты без отображения на соответствующей карте противоречит условиям использования.

В итоге я обратился к готовому агрегатору https://zaprauka.by/map/. Этот сайт предоставляет json api. Есть возможность получить список заправок, розничных продавцов и регионов.

В программе через HttpClient запрашивается информацию о станциях, регионах и организациях. Далее сведения о станциях дополняются и выводятся.

Программа предоставляет интерфейс для вывода списка заправок на экран или в заданный файл.
run with:\n-c to show result in console;\n-f [filename] to write result in file;
